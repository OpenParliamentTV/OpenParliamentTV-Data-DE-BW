[OpenParliamentTV-Tools](https://github.com/OpenParliamentTV/OpenParliamentTV-Tools):
[`optv/parliaments/DE-BW`](https://github.com/OpenParliamentTV/OpenParliamentTV-Tools/blob/main/optv/parliaments/DE-BW).
